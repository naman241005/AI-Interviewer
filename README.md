# AI Interviewer

An AI-powered voice interview system built using Python and Streamlit.

## Features
- AI-generated interview questions
- Voice interaction
- Resume/interview analysis
- Real-time feedback
- Report generation

## Tech Stack
- Python
- Streamlit
- Groq API
- OpenAI
- WebRTC
- Speech Recognition

## Installation

```bash
git clone REPO_LINK
cd AI-Interviewer
pip install -r requirements.txt
streamlit run app.py
```
